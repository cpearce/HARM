HARM - Streaming Association Rule Miner
Licensed under a Apache 2 style license.

HARM is an association rule miner. It finds associations in transaction data
sets using multiple methods. 

HARM is written in C++, and currently only builds on Windows with Visual Studio
2012 or later, or MacOSX via Makefile.macosx.

General guidelines for scientific programming can be found here:
http://www.plosbiology.org/article/info:doi/10.1371/journal.pbio.1001745
